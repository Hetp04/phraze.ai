# Phraze AI

A AI-powered web annotation tool with chat interface.

## Quick Start

1. **Clone & Install**
   ```bash
   git clone https://github.com/Hetp04/phrazeai.git
   cd phrazeai
   npm install
   ```

2. **Run the App**
   ```bash
   npm run dev
   ```

## Project Structure

```
phrazeai/
├── src/           # React app
├── extension/     # Chrome extension
└── styles/        # CSS files
```

## Development

- `npm run dev` - Start development server
- `npm run build` - Build for production
